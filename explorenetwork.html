Networking Fundamentals

These study blocks make the topic easier to revise by grouping together the most important ideas beginners usually struggle with first.

Core Concepts

IP Address An IP address is the number used to identify a device on a network so data knows where to go.
Subnet Mask A subnet mask helps separate the network portion of an IP address from the host portion.
Default Gateway The default gateway is the router a device uses to reach networks outside its own local network.
DHCP DHCP automatically gives devices IP settings instead of requiring manual setup.
MAC Address A MAC address is a hardware-level identifier tied to a network interface.

Why Networking Matters

Everything connects through networks Web browsing, cloud services, messaging, video calls, and shared systems all depend on networking.
It supports many IT careers Networking knowledge is useful in support, cybersecurity, cloud, systems administration, and DevOps.
It improves troubleshooting Understanding basic networking helps you solve slow internet, name resolution, and connectivity problems faster.
It links into security Good network knowledge helps you understand secure access, segmented networks, and protected communication.
It is a strong foundation topic Once networking becomes clear, many other technical topics become much easier to understand.

Quick Networking Facts

Use these fast reminders as mini revision points before going into quizzes or flashcards.

Router

Connects different networks and forwards traffic between them.

Switch

Connects devices inside the same local network.

DNS

Converts names like websites into IP addresses.

DHCP

Automatically assigns network settings to devices.

TCP

Reliable protocol that checks order and delivery of data.

UDP

Faster protocol that does not focus on guaranteed delivery.
