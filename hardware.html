Core Concepts

CPU The CPU processes instructions and performs the main calculations and operations in a computer.
RAM RAM is temporary memory used by active programs and data while the system is running.
SSD An SSD stores files and the operating system permanently, and it is usually faster than an HDD.
Motherboard The motherboard is the main board that connects and supports the system’s major components.
PSU The power supply unit converts electricity into the form needed by the internal hardware parts.

Why Hardware Matters

It is the physical foundation All software, networks, and systems depend on working physical devices underneath them.
It improves troubleshooting Understanding hardware helps users diagnose slow performance, boot failure, faulty devices, and storage problems.
It supports IT roles Hardware knowledge is useful in support, systems administration, networking, and general IT operations.
It helps with upgrades Knowing hardware makes it easier to understand storage upgrades, RAM changes, peripherals, and compatibility.
It connects to everything else Hardware understanding makes operating systems, networking, and troubleshooting much easier to follow.

Quick Hardware Facts

Use these short reminders as mini revision notes before moving into quizzes, flashcards, or deeper study.

CPU

Executes instructions and handles system processing.

RAM

Temporary working memory used by active programs.

SSD

Fast permanent storage for files and the operating system.

HDD

Traditional storage device that usually offers larger capacity.

Motherboard

Main board that connects major internal components.

PSU

Supplies electrical power to the computer’s internal parts.
