Cloud Computing

Understanding on-demand computing, virtualization, and global infrastructure.

The Core Pillars

Why businesses are moving away from physical hardware to the cloud.

On-Demand Self-Service

Users can provision computing capabilities, such as server time and network storage, as needed automatically without requiring human interaction with each service provider.

Resource Pooling

The provider's computing resources are pooled to serve multiple consumers using a multi-tenant model, with different physical and virtual resources assigned according to demand.

Rapid Elasticity

Capabilities can be elastically provisioned and released, in some cases automatically, to scale rapidly outward and inward commensurate with demand.

Service Models (IaaS vs PaaS vs SaaS)

Model You Manage Provider Manages
IaaS (Infrastructure) Applications, Data, OS Virtualization, Servers, Storage
PaaS (Platform) Applications & Data OS, Middleware, Runtime
SaaS (Software) Nothing (Use the app) Everything
